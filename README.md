 ATM Simulator in c++

## Description
This project was made by me without the use of AI, what it does is it simulates how an ATM works with live transactions, deposits and even with warnings when getting close to being in debt.

## Tools/Libraries Used
- Codeblocks
- iostream library
- windows.h library

## How to run
1. Open ATM simulator.cpp in your IDE (codeblocks, VS code)
2.Compile the program.
3. You will be able to simulate
