# ATM-Simulator-C-

## Description
A code made with C++ simulating how an ATM works, without the use of AI. One of the first coding projects i have ever made, it has real life ATM features such as
deposits, transactions and even warnings when near debt.

##Tools/Libraries Used
-Codeblocks IDE
-iostream library
-windows.h library

##How to run
1.Open `ATM simulator.cpp` in your ide(codeblocks, visual studio code)
2.Compile and run the program.
3.Enter your choices when prompted, and the program will simulate based off your choices.
